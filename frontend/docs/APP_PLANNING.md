# 한자능력검정시험 5급 준비 앱 기획서

## 1. 앱 개요

### 1.1 목적
- 한자능력검정시험 5급 준비를 위한 학습 및 연습 앱
- 초·중학생을 대상으로 한 쉽고 재미있는 학습 경험 제공

### 1.2 핵심 가치
- **학습**: 한자의 뜻, 음, 획순, 예문을 체계적으로 학습
- **연습**: 퀴즈와 게임을 통한 반복 학습
- **평가**: 시험 모드를 통한 실전 대비

---

## 2. 앱 구조

```
한자 5급 앱
├── 홈 화면
│   ├── 학습 시작
│   ├── 단원 선택
│   └── 진도 현황
├── 학습 모드
│   ├── 한자 카드 뷰
│   ├── 상세 정보 (뜻, 음, 획순, 예문)
│   └── 학습 진행률
├── 연습 모드
│   ├── 퀴즈 모드 (4지선다)
│   ├── 게임 모드 (맞추기 게임)
│   └── 결과 및 오답 노트
├── 시험 모드
│   ├── 제한 시간 설정
│   ├── 문제 풀이
│   └── 결과 및 성적표
└── 통계/진도
    ├── 학습 진도
    ├── 정답률 통계
    └── 약점 한자 분석
```

## 3. 주요 화면 목록

### 3.1 홈 화면 (Home)
**목적**: 앱의 메인 진입점, 빠른 접근 제공

**기능**:
- 최근 학습 단원 표시
- 학습 진도 요약 (진행률, 완료 단원 수)
- 빠른 시작 버튼 (이어서 학습하기)
- 메뉴 접근 (학습, 연습, 시험, 통계)

**UI 요소**:
- 상단: 앱 로고/제목
- 중앙: 진도 원형 차트 또는 진행 바
- 하단: 주요 기능 버튼 그리드

---

### 3.2 단원 선택 화면 (Chapter Selection)
**목적**: 학습할 단원 선택

**기능**:
- 단원 목록 표시 (예: 1단원 ~ 10단원)
- 각 단원별 한자 개수 표시
- 단원별 완료 상태 표시 (완료/진행중/미시작)
- 단원별 난이도 표시 (선택사항)

**UI 요소**:
- 단원 카드 리스트
- 각 카드: 단원명, 한자 개수, 완료율, 잠금 상태

---

### 3.3 학습 화면 (Study)
**목적**: 한자 상세 학습

**기능**:
- 한자 카드 플립 애니메이션 (앞면: 한자, 뒷면: 정보)
- 한자 정보 표시:
  - 한자 (큰 글씨)
  - 음 (훈음)
  - 뜻
  - 획순 (애니메이션 또는 정적 이미지)
  - 예문 (2-3개)
- 이전/다음 한자 이동
- 학습 완료 체크
- 즐겨찾기 추가

**UI 요소**:
- 상단: 단원명, 진행률 바
- 중앙: 한자 카드 (터치로 플립)
- 하단: 이전/다음 버튼, 완료 체크박스

---

### 3.4 연습 모드 - 퀴즈 화면 (Quiz)
**목적**: 4지선다 문제를 통한 연습

**기능**:
- 문제 유형:
  - 한자 → 뜻 맞추기
  - 뜻 → 한자 맞추기
  - 한자 → 음 맞추기
  - 음 → 한자 맞추기
- 문제 랜덤 출제
- 즉시 피드백 (정답/오답)
- 오답 노트 자동 저장
- 점수 및 진행률 표시

**UI 요소**:
- 상단: 문제 번호, 점수
- 중앙: 문제 (한자 또는 뜻)
- 하단: 4지선다 선택지 버튼
- 결과 팝업: 정답 표시, 다음 문제 버튼

---

### 3.5 연습 모드 - 게임 화면 (Game)
**목적**: 재미있는 게임을 통한 학습

**게임 유형**:
1. **맞추기 게임**: 한자와 뜻을 짝 맞추기
2. **타이핑 게임**: 뜻을 보고 한자 입력
3. **순서 맞추기**: 획순 순서 맞추기

**기능**:
- 시간 제한 (선택사항)
- 점수 시스템
- 콤보 보너스
- 게임 오버 시 결과 표시

**UI 요소**:
- 게임 영역 (게임 유형에 따라 다름)
- 점수, 시간 표시
- 게임 오버 모달

---

### 3.6 시험 모드 화면 (Exam)
**목적**: 실전 시험 대비

**기능**:
- 제한 시간 설정 (기본 30분)
- 문제 출제 (랜덤 또는 단원별)
- 타이머 표시
- 문제 건너뛰기 (나중에 다시 풀기)
- 제출 전 확인
- 결과 화면:
  - 점수
  - 정답률
  - 소요 시간
  - 오답 리스트

**UI 요소**:
- 상단: 타이머, 문제 번호
- 중앙: 문제 및 선택지
- 하단: 이전 문제, 다음 문제, 제출 버튼
- 결과 모달: 성적표

---

### 3.7 통계/진도 화면 (Statistics)
**목적**: 학습 현황 파악

**기능**:
- 전체 진도율 표시
- 단원별 완료율
- 정답률 통계 (문제 유형별)
- 약점 한자 리스트
- 학습 시간 통계
- 성적 그래프 (시험 모드 결과)

**UI 요소**:
- 진도 차트 (원형 또는 막대)
- 통계 카드들
- 약점 한자 리스트
- 그래프 (선택사항)

---

### 3.8 오답 노트 화면 (Wrong Answers)
**목적**: 틀린 문제 복습

**기능**:
- 오답 한자 리스트
- 오답 횟수 표시
- 오답 문제 다시 풀기
- 오답 노트에서 제거

**UI 요소**:
- 오답 한자 카드 리스트
- 각 카드: 한자, 오답 횟수, 복습 버튼

## 4. 데이터 구조

### 4.1 한자 데이터
```typescript
interface Hanja {
  id: string;
  character: string;      // 한자
  sound: string;          // 음
  meaning: string;        // 뜻
  strokeOrder: string[];  // 획순 (이미지 경로 또는 좌표)
  examples: Example[];    // 예문
  chapter: number;        // 단원
  difficulty: number;     // 난이도 (1-5)
}

interface Example {
  sentence: string;       // 예문
  meaning: string;        // 예문 뜻
}
```

### 4.2 학습 진행 데이터
```typescript
interface Progress {
  userId: string;
  chapter: number;
  completedHanja: string[];  // 완료한 한자 ID 리스트
  studyCount: number;        // 학습 횟수
  lastStudied: Date;
}

interface QuizResult {
  hanjaId: string;
  questionType: string;
  isCorrect: boolean;
  timestamp: Date;
}

interface ExamResult {
  examId: string;
  score: number;
  totalQuestions: number;
  correctAnswers: number;
  timeSpent: number;
  wrongAnswers: string[];
  timestamp: Date;
}
```

## 5. UX/UI 가이드라인

### 5.1 디자인 원칙
- **단순성**: 복잡한 메뉴 최소화
- **명확성**: 큰 글씨, 명확한 버튼
- **즉각적 피드백**: 정답/오답 즉시 표시
- **시각적 재미**: 애니메이션, 게이미피케이션 요소

### 5.2 모바일 우선 설계
- 터치 친화적 버튼 크기 (최소 44x44px)
- 스와이프 제스처 활용 (카드 넘기기)
- 세로 모드 최적화
- 반응형 레이아웃

### 5.3 색상 및 타이포그래피
- **주 색상**: 학습에 집중할 수 있는 차분한 색상
- **강조 색상**: 정답(초록), 오답(빨강)
- **한자 폰트**: 명확하게 보이는 한자 전용 폰트
- **가독성**: 충분한 대비, 적절한 줄간격

## 6. 기술 스택 제안

### 6.1 프론트엔드
- **React 18+**: UI 라이브러리
- **TypeScript**: 타입 안정성
- **React Router**: 라우팅
- **Zustand / Context API**: 상태 관리
- **Tailwind CSS / Styled Components**: 스타일링
- **Framer Motion**: 애니메이션

### 6.2 데이터 관리
- **LocalStorage / IndexedDB**: 로컬 데이터 저장
- **JSON 파일**: 한자 데이터 (초기 버전)

### 6.3 빌드 도구
- **Vite**: 빠른 개발 서버 및 빌드
- **ESLint + Prettier**: 코드 품질
