# 한자능력검정시험 5급 준비 앱

한자능력검정시험 5급을 준비하는 학생들을 위한 학습 앱입니다.

## 주요 기능

- 📚 **학습 모드**: 한자의 뜻, 음, 획순, 예문을 체계적으로 학습
- ✏️ **퀴즈 모드**: 4지선다 문제를 통한 연습
- 🎮 **게임 모드**: 재미있는 게임을 통한 학습 (준비 중)
- 📝 **시험 모드**: 실전 시험 대비 (준비 중)
- 📊 **통계**: 학습 진도 및 정답률 확인
- 📖 **오답 노트**: 틀린 문제 복습

## 기술 스택

- React 18
- TypeScript
- React Router
- Zustand (상태 관리)
- Framer Motion (애니메이션)
- Vite (빌드 도구)

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── Layout/         # 레이아웃 컴포넌트
│   └── HanjaCard/      # 한자 카드 컴포넌트
├── pages/              # 페이지 컴포넌트
│   ├── Home/           # 홈 화면
│   ├── ChapterSelection/ # 단원 선택
│   ├── Study/          # 학습 화면
│   ├── Quiz/           # 퀴즈 화면
│   ├── Game/           # 게임 화면
│   ├── Exam/           # 시험 화면
│   ├── Statistics/     # 통계 화면
│   └── WrongAnswers/   # 오답 노트
├── store/              # 상태 관리
├── types/              # TypeScript 타입 정의
├── data/               # 데이터 파일
├── utils/              # 유틸리티 함수
└── App.tsx             # 메인 앱 컴포넌트
```
